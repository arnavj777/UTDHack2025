# Component Copy Instructions

The components from Figmauitest-main need to be copied to frontend/src/components.

## Manual Copy Steps

1. Copy all files from `Figmauitest-main/src/components/` to `frontend/src/components/`
2. Copy all files from `Figmauitest-main/src/styles/` to `frontend/src/styles/`

## Files to Copy

- All .tsx files in Figmauitest-main/src/components/
- All files in Figmauitest-main/src/components/ui/
- All files in Figmauitest-main/src/components/figma/
- All files in Figmauitest-main/src/styles/

## Note on Imports

Some component files may have versioned imports like `@radix-ui/react-slot@1.1.2` - these should be changed to `@radix-ui/react-slot` (remove version numbers).
